# Sales dashboard wallmart

[Wallmart Sales Dasboard](https://public.tableau.com/views/Walmart_Sales_16790866143180/Dash_CARATULA?:language=es-ES&:display_count=n&:origin=viz_share_link)

A dashboard project that make a sales analysis developed in Tableau using a dataset with Walmart sales information extracted from www.kaggle.com.

## Overview

In this section you can see a brief summary of the dashboards available in this project.

### Supermarket sales analysis

This dashboard includes a bar chart, a pie chart, KPIs, and a filter where you can choose the data according to category, state, city, product name, and the top 10 best-selling products.

### Profit 

Dashboard for profits based on product categories, including a filter to view data by category. Another chart displays profits along with the total over the years of analysis.
